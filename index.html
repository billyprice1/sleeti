Installation

Prerequisites

  • nginx/Apache 2
  • PHP 7
  • MySQL server
  • Composer

Setup

  • git clone
  • cd
  • composer install
  • Create database and source sleeti.sql from mysql console
  • Install webserver configs
  • Restart nginx/Apache
  • Visit website (/install)
  • Fill out fields
  • Enjoy!
